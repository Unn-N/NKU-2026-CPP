# 基于视觉感知与人机交互的高准确率排序算法
## 简介
 
本项目为南开大学2026高级语言设计C++课程实训作业，

基于 Qt 平台 + C++ 开发，是一款集排序算法可视化、人机交互手动排序于一体的图形化小程序。

项目设计灵感源自油管博主swap.开发的优质算法可视化交互网站 swapjs.dev

借鉴其趣味化、沉浸式的排序交互理念，摒弃传统枯燥的代码推演模式，以视觉可视化+手动操作的形式，直观呈现各类排序算法的底层逻辑与运行过程。

多次迭代优化交互细节、视觉效果与算法逻辑（真的很多次，真的很细节）

## 下载与安装

本项目已打包发布正式版本，无需复杂编译配置，可直接前往项目 Release 页面下载安装包，快速体验程序。

## 使用说明

程序核心包含两大功能模块，全方位满足算法学习与交互体验需求：

### 1. 排序算法可视化演示
   
自动运行各类经典排序算法，以图形化界面动态展示数据排序全过程，清晰呈现算法原理、执行步骤与排序效率差异，适合入门学习排序算法。

### 2. 手动交互式排序体验
   
支持用户手动操控数据完成排序，模拟算法排序逻辑，通过人机交互的方式，深度理解排序规则与核心逻辑，提升对排序算法的感知与掌握程度。

本人深知这是一个功能十分简单，深度不足的程序，所以在交互细节和视觉效果等方面十分严格，努力雕琢程序的质感，尽最大努力让交互体验让人满意，让这个程序脱离“粗糙”一词。致力于让所有用户都体验到丝滑顺畅的交互体验
